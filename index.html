Projects

Mapping Economic Bias in the NBA

The Problem: It is often the case that econometric analyses often conclude with a wall of coefficients, a static list that confirms bias exists. This is technically correct, but it fails to reveal the underlying bias structure and how each player fits into that structure. Furthermore, applying standard models to the NBA leads to severe misspecification because deterministic contracts (Rookie Scale, Max Contract) distort free-market signals.

The Solution: I developed a Stratified Double Machine Learning framework. By strictly isolating free-market players to learn the true "price" of bias, I could counterfactually apply these valuations to the full league. I then fed these attributions into my own probabilistic dimension reduction model to interactively visualize the geometric topology of market inefficiency. Practically, this means we can visualize exactly how and where the NBA market systematically misprices players, not just that bias exists. On a side, my models suggest that Lebron James's compensation is strongly upward biased because of his seniority, accounting for performance.

End-to-End Flight Forecasting System

The Problem: If you search online, you will not find many historical flight listings. You also will not find any many usful predictive tools.

The Solution: I architected a complete cloud-native system on AWS. I built the fully serverless ETL pipeline using Dockerized Lambda to automate data collection and construct a data lake in S3. Then, I developed a "Dual-Model" architecture: a regression model for market-level forecasting and a classification model to generate tactical "Buy/Wait" signals for individual flights.

Interactive Causal Discovery App

The Problem: Being honest, I didn't really have a problem I wanted to solve going in. I was just always curious about the phrase "Correlation does not imply Causation". I always wondered "Ok, but what can we say about causality, if at all?" This project were the highlights of exploring that question. I used Elements of Causal Inference by Jonas Peters, Dominik Janzing, and Bernhard Schölkopf as my textbook.

The Solution: I built a interactive web application to simulate the PC Algorithm from scratch. It allows users to visualize how conditional independence testing can be used to recover causal graphs from observational data, demonstrating both the limitations and power of causal discovery.
